About HackUnion

HackUnion is a student-driven builder community dedicated to helping learners turn ideas into real-world projects through collaboration, mentorship, and hands-on learning.
Founded by Afnan Abdul Vasay to create a culture of innovation and practical skill development, HackUnion provides structured build sprints, workshops, hackathons, and peer-led sessions that encourage students to learn by building together.

Builders Guild

Intensive Building Sprint

6-Hour focused building sprints where builders ship real projects with daily standup and mentor support.

Event Details

Duration: 6 hours

Format: Daily commits, code reviews, mentor checkpoints
