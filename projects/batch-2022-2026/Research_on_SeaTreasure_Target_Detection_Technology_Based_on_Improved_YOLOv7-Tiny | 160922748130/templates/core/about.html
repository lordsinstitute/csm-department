About SeaTreasure AI

An advanced underwater sea treasure detection platform powered by an improved UPA-YOLOv7-Tiny deep learning architectue.

Mission

The primary goal of this project is to provide a highly accurate, real-time object detection system for underwater environments. Detecting marine resources like sea cucumbers, sea urchins, and shells is critical for marine biology research, resource management, and ecological monitoring.

Underwater imagery presents unique challenges including severe color distortion, low contrast, blurriness, and light attenuation. Our platform leverages an improved YOLOv7-Tiny network featuring an Underwater Perception Attention (UPA) module designed specifically to combat these issues.

Key Innovations

Lightweight Backbone Architecture

The model replaces standard convolutions with FasterNet blocks adopting Partial Convolutions (Pconv). This drastically reduces floating-point operations (FLOPs) and inference latency, achieving speeds suitable for edge devices.

Underwater Perception Attention (UPA)

By integrating Spatial Sub-pixel Convolution and channel attention mechanisms, the model effectively extracts robust multiscale features, filtering out underwater noise like floating sand and improving small-object detection (e.g., shells).

WIoUv3 Loss Function

We replaced standard CIoU with WIoUv3, which dynamically allocates weights based on bounding box overlap quality. This prevents the model from being over-penalized by low-quality samples, accelerating convergence and boosting mAP.
