## 📊 How to Use This Dataset

### Testing Procedure:

1. **Copy a question** from this file
2. **Go to the AI Chat**: http://127.0.0.1:8000/interaction/
3. **Select emotional state** (shown in parentheses for realistic response variation)
4. **Paste the question** into the "Your Message" field
5. **Click "Send Message"**
6. **Review the response**:
   - Does it address the question?
   - Is the reasoning clear?
   - Does the confidence score make sense?
   - Is the tone appropriate to the emotional state?

## 📌 Pro Tips for Testing

1. **Try Different Emotional States**: Same question with different emotions gives varied responses
2. **Test Fallback Responses**: Questions will work even without OpenAI API key
3. **Check Conversation History**: Verify all messages appear in chat history
4. **Test Quick Actions**: Use the quick action buttons at the bottom
5. **Monitor Confidence Scores**: Responses should have 70-95% confidence
6. **Review Reasoning**: Each response should have a "Reasoning" section
